# Meu Site Acessível

Este é um site criado para praticar HTML, CSS e JavaScript, com foco em acessibilidade digital.  
O projeto contém:
- Navegação simples
- Contraste ajustável
- Leitura de texto por voz

## Como visualizar
Basta abrir o arquivo `index.html` em qualquer navegador.

## Tecnologias usadas
- HTML5
- CSS3
- JavaScript
